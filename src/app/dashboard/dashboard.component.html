Project Overview:
This project was created by Isaak Treat and Jen Macklin.
It contains two main components:
- An Astronomy Picture of the Day
- Martian Weather Tracker for the Past 7 Sols
All data used for both components are property of NASA
We have been given use to display it under their permission at api.nasa.gov Project Description:
Astronomy Picture of the Day:
Each day, a new astronomy photo is displayed.
Along with the photo, you're able to find the:
- Title
- Author
- Description
- Copyright

Mars Weather:
Mars Weather displays the past 7 Sols and their temperature statistics on each Sol.
Clicking on a Sol will give a greater insight into what each Sol was like, including:
- Temperature High and Low
- Atmospheric Pressure
- Wind Speeds
- Season
